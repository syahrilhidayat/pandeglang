Jawaban

1. a-Web adalah sistem pengiriman dokumen tersebar yang berjalan di internet
b- Portal website yang menyediakan beraneka ragam informasi untuk para pengunjungnya
c- Blog merupakan catatan harian yang ditayangkan secara online

2.web browser adalah perangkat lunak yang digunakan untuk menyajikan sebuah informasi yang ada didalam jaringan internet. Contoh web browser yang sering digunakan oleh pengguna internet, diantaranya, Mozilla Firefox, Safari, Opera, Google Chrome, dan UC Browser.

3. menerima dan memberikan respon permintaan yang dikirim user melalui browser berupa halaman web yang merupakan dokumen HTML.

4. Protocol HTTP melalui port 80 pada server akan melewatkan alamat yang diketik melalui browser. Alamat unik situs internet diketik pada baris URL di halaman browser. Browser memiliki aturan yang disepakati sebelumnya dalam hal ini protocol dengan menggunakan TCP/IP. Web browser yang telah di install pada user (client) akan menampilkan dokumen atau informasi dengan terlebih dahulu menterjemahkan (interpreting) dari bahasa atau script dalam bentuk HTML kedalam bentuk teks atau informasi yang ditampilkan di browser dan dapat dibaca atau diakses oleh pengguna. Tampilan dapat berupa teks, gambar, video, suara atau lainnya.
